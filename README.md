# cnwebs-web-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及网络资源导航、信息整理与指南类文档，旨在以静态页面的形式提供参考和访问便利。

仓库不针对任何特定域名或网站，所有页面均为独立构建，便于直接查看或部署到静态托管服务。

## 目录说明

```
├── README.md           # 本文件
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：核心内容目录，每个 HTML 文件为一个独立页面，可单独访问。
- `assets/`：存放页面共用的样式、脚本及图片资源，便于统一维护。

## 页面归档说明

- 每个页面均为独立、完整的 HTML 文件，不依赖外部数据库或后端服务。
- 页面内容以信息整理、操作指南或资源索引为主，风格简洁直接。
- 页面之间无强制关联，可独立浏览或部署。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进页面内容或修复错误。
- 新增页面请放入 `pages/` 目录，并保持命名清晰。
- 若使用公共资源，请确保文件路径正确，并尽量复用 `assets/` 下的文件。
- 本仓库遵循开源协议（见 LICENSE 文件），请在使用时遵守相关条款。
